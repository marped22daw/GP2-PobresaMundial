Sobre nosotros

Los desarrolladores de este sitio web somos Marc Pedemonte, Pol Sanz y Angel Torres. Este proyecto comenzó durante nuestra formación en DAW y recientemente hemos decidido mejorarlo. El objetivo era realizar un proyecto junto a un equipo utilizando el framework de Angular. Este sitio web consiste en mostrar gráficamente datos de países de todo el mundo mediante tablas con datos y gráficos que permiten visualizar mejor las estadísticas entre países.
